Addendum
  • The game revenge of lights out is designed using HTML, CSS and Java Script
  • The javascript to the website is linked using 'script' tag and style using 'style' tag
  • Initially, the board has some randomly generated black tiles and, the user tries to turn the game board white
  • The number of clicks the user makes is counted and displayed
  • When the user turns the board white an alert box is displayed along with the count
  • The game is initialised by calling the initializeGame funciton which sets the game board
  • onClick of each tile toggleLights is called which then calls incrementCounter, toggleState, updateLights and checkWin
  • incrementCounter increments the number of moves
  • toggleState again calls toggleSingleStatebased (which changes the color) on various conditions
  • Check win checks if the board has completely turned white and displays an alert box
    • Problems faced:
  • Creating elements using java script and appending them to parent was a challege
  • Couldn't fetch elements using getElementByClassName
  • Cannot read properties of null was always the error I saw
  • Java script added elements to existing elements
  • Styling was a challege
    • Solution:
  • Used debugger to find out the reason for code break
  • Replaced getElementByClassName by getElementById to get Elements
  • Cleared out existing elements using a condition and replacing it with empty string

  • Learnt various javascript functions and dynamic screen updating
